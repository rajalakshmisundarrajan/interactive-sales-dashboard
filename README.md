# 📊 Interactive Sales Dashboard

An interactive data visualization dashboard built using Python to analyze sales performance, customer behavior, and product trends.



## 🚀 Features

* 📈 Sales trend analysis over time
* 🛍️ Product performance visualization
* 👥 Customer segmentation insights
* 📊 Multiple chart types (Seaborn, Matplotlib, Plotly)
* 🎨 Clean and professional dashboard layout
* ⚡ Interactive visualizations (if using Plotly/Streamlit)



## 🛠️ Tech Stack

* Python
* Pandas – Data processing
* NumPy – Numerical operations
* Seaborn & Matplotlib – Statistical visualization
* Plotly – Interactive charts
* Streamlit (optional) – Web-based dashboard



## 📂 Project Structure

```
interactive-sales-dashboard/
│
├── dashboard.py          # Main dashboard script
├── sales_data.csv        # Dataset
├── README.md             # Project documentation
└── visualizations/       # Output charts (optional)
```

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/interactive-sales-dashboard.git
cd interactive-sales-dashboard
```

2. Install dependencies:

```
pip install pandas numpy seaborn matplotlib plotly streamlit
```



## ▶️ How to Run

### Run as Python script:

```
python dashboard.py
```

### Run as Streamlit app (Recommended):

```
streamlit run dashboard.py
```
